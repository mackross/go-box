go-box
=======

go-box provides a client for [Box's 2.0 API](http://developers.box.com/docs/).

It is still a work in progress.

The supported features include

    * Oauth2 authorization
    * Following folder operations:

        * Get
        * Create
        * Delete
        * Rename
        * Move
        * Copy
        * Share
        * Unshare
